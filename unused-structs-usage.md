# Unused Structs Usage

Documents what unused space in Dwarf Fortress' structs this mod occupies.

- Squad room mode bit 31: stores whether the room building is a linked turret.
- Building machine info flags bits 26 to 31 (for turrets): stores an number for the turret firing cooldown.
- Wound flags and wound part flags2 bits 25 to 31: stores information about pain level.
